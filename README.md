# Nexus AI — B2B SaaS Event Management Platform

Nexus AI — это полноценная B2B SaaS-платформа (Telegram бот) для интеллектуального сорсинга и управления персоналом на мероприятиях. Система поддерживает многозадачность (multi-tenancy), позволяя владельцу продавать доступ рекрутерам различных компаний по подписке.

## 🌟 Основные возможности

### 👑 Для Владельца Платформы (Super Admin):
- **Управление компаниями**: Создание компаний, установка ежемесячной платы.
- **Управление рекрутерами**: Привязка рекрутеров к конкретным компаниям.
- **Контроль подписок**: Продление подписки на N месяцев, автоматическая блокировка при истечении.
- **Изоляция данных**: Каждая компания работает в своем изолированном пространстве (Multi-tenancy).
- **Кастомные группы**: Привязка отдельного Chat ID группы для опросов каждой компании.

### 👨‍💼 Для Рекрутеров:
- **Управление мероприятиями**: Создание ивентов, задание лимитов и требований.
- **Опросы в группах**: Публикация опросов в привязанную группу компании.
- **Интерактивный отбор**: Выбор кандидатов через удобный интерфейс.
- **Упрощенное назначение времени**: Быстрая установка смен через кнопки и простые текстовые сообщения (без сложных команд).
- **Google Таблицы & Excel**: Генерация отчетности с авто-подсчетом выплат.
- **Аудит**: Полная история действий по каждому мероприятию.

### 🙋‍♂️ Для Кандидатов:
- Простая регистрация (роль, пол, контакт).
- Голосование в опросах и получение приглашений.
- Подтверждение участия кнопкой в боте.

## 🛠 Технологический стек
- **Python 3.10+ / python-telegram-bot**
- **Supabase (PostgreSQL)** — хранение данных и RLS-политики.
- **APScheduler** — фоновые задачи и напоминания.
- **Google Sheets API** — интеграция с онлайн-таблицами.
- **Openpyxl** — экспорт в .xlsx.

## 🚀 Быстрый старт

1. **Установка зависимостей:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Настройка `.env`:**
   Заполните `BOT_TOKEN`, `SUPABASE_URL`, `SUPABASE_KEY` и, самое главное, `SUPER_ADMIN_ID` (ваш Telegram ID).

3. **База данных:**
   Выполните SQL-скрипт `supabase_schema.sql` в Supabase SQL Editor.

4. **Запуск:**
   ```bash
   python main.py
   ```

## 📜 Команды управления

### Владелец:
- `/owner` — Главная панель управления SaaS-платформой.

### Рекрутер:
- `/create_event` — Создать мероприятие.
- `/list_events` — Список мероприятий и управление (опросы, отбор, экспорт).
- `/set_times` — Интерактивное назначение времени.
- `/help` — Справка по всем командам.
