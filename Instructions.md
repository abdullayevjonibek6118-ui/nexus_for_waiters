# Инструкция для владельца Nexus AI (SaaS)

Поздравляем! Ваш бот теперь является B2B платформой. Ниже описаны шаги, которые вам нужно предпринять для запуска и управления бизнесом.

## 1. Первоначальная настройка
1. Откройте файл `.env`.
2. Найдите переменную `SUPER_ADMIN_ID`.
3. Впишите туда ваш цифровой ID Telegram (можно узнать у @userinfobot).
4. Перезапустите бота.

## 2. Управление через `/owner`
Введите команду `/owner` в боте. Вам откроется панель управления:

### Создание компании
1. Нажмите **«Создать компанию»**.
2. Введите название (например, "Ресторан Прага").
3. Укажите ежемесячную стоимость подписки.

### Настройка группы для опросов
1. Перейдите в **«Список компаний»** -> Выберите компанию.
2. Нажмите **«Указать ID группы»**.
3. Введите ID группы, в которую рекрутер этой компании будет постить опросы (ID начинается с `-100...`).
   *Чтобы узнать ID группы, добавьте в нее бота и напишите `/start`, бот выведет ID в логи или используйте специальные боты.*

### Добавление рекрутера
1. В меню компании нажмите **«Добавить рекрутера»**.
2. Введите Telegram ID рекрутера, которому вы продали доступ.
3. Теперь этот человек при нажатии `/start` увидит интерфейс рекрутера этой компании.

### Продление подписки
1. В меню компании нажмите **«Продлить подписку»**.
2. Выберите количество месяцев. Бот автоматически обновит дату, до которой рекрутеры этой компании смогут работать.

## 3. Регистрация кандидатов
Просто отправьте ссылку на бота кандидатам. Они должны:
1. Нажать `/start`.
2. Выбрать **«Я Кандидат»**.
3. Указать пол и поделиться контактом.
4. После этого они смогут участвовать в опросах, опубликованных в группах.

## 4. Упрощенное назначение времени
Когда рекрутер отобрал кандидатов через `/list_events` -> **«Управление»** -> **«Отобрать»**, он переходит к назначению времени:
1. Команда `/set_times <id>`.
2. Появится список официантов с кнопками.
3. Рекрутер нажимает кнопку (например, «Назначить всем») и просто пишет время: `09:00 20:00`.
4. Бот сам обновит базу и подготовит данные для экспорта.
